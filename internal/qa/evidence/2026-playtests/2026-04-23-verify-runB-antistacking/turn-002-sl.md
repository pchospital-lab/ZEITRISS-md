# Turn 2 — SL

```
Kodex: Level-Up ausstehend — Save nach Wahl.
```

---

**Stopp — nur eine Wahl pro Stufe.** [7]

Pro Stufenaufstieg ist genau **eine** Option erlaubt: `+1 Attribut` **oder** `Talent/Upgrade` **oder** `+1 SYS`. Kein Stacking, keine Kombination. Das ist eine harte Regelgrenze [7].

Du musst dich entscheiden:

---

**`[1] +1 INT`**
INT 5 → 6. Bessere Technik-, Hacking- und Spurenanalyse-Proben.
*(Kein Würfeltypwechsel — W10 erst ab INT 11)*

**`[2] Tatortanalyse → Fortgeschritten`**
+3 statt +2 auf Tatortanalyse-Proben. Dazu: Reroll 1×/Mission.
*(Dein vorhandenes Talent wächst — solider Upgrade für SPLINTERs Kernkompetenz)*

**`[3] +1 SYS`**
SYS 3 → 4. Ein zusätzlicher Implantat-Slot.

**`[4] Freie Aktion`**
Andere Idee oder Rückfrage.
