# Uploads-Ablage

Dieser Ordner ist eine **Eingangsablage** für externe Review-/Upload-Artefakte.

## Arbeitsregel

1. Upload-Artefakt hier ablegen.
2. Relevante Punkte in `internal/qa/plans/ZEITRISS-qa-fahrplan-2025.md` und
   `internal/qa/logs/2025-beta-qa-log.md` überführen.
3. Artefakt danach als "Archivhinweis" markieren (oder in interne QA-Evidence
   verschieben), damit der operative Status nur noch in den QA-Dokumenten lebt.

Damit bleibt klar: **Uploads sind Quelleingang**, die verbindliche Umsetzung und
Nachverfolgung passiert unter `internal/qa/`.
